## 🚀 Overview

This portfolio is designed to be lightning-fast, accessible, and clean. It uses a modern **Light Mode** aesthetic with crisp typography and subtle interactions. 

Unlike many modern portfolios that rely on heavy JavaScript frameworks, this project is built entirely with **Vanilla HTML, CSS, and JavaScript** to ensure maximum performance, instant load times, and perfect SEO scores.

## 🛠️ Tech Stack

- **HTML5**: Semantic structure for accessibility and SEO.
- **CSS3**: Custom variables (`:root`), flexbox/grid layouts, and responsive media queries.
- **Typography**: [Archivo](https://fonts.google.com/specimen/Archivo) for headings and [Space Grotesk](https://fonts.google.com/specimen/Space+Grotesk) for body text.
- **Icons**: SVG paths integrated directly into the HTML to eliminate external font-icon network requests.

## ⚙️ Running Locally

Because this project uses vanilla web technologies, no build step or node package installation is required! 

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Greekkgod/portfolioo.git
   ```
2. **Open the project:**
   Simply double-click `index.html` to open it in your browser.
3. **For Live Reloading (Optional):**
   If you are editing the code and want to see changes live, you can use the VS Code [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) extension, or run a simple Python server:
   ```bash
   python -m http.server 8000
   ```
   Then navigate to `http://localhost:8000`.
